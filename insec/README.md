。2. 数学本质：两条射线的“异面交点”
根据共线方程，相机中心、照片上的点、地面上的真实点，三点一线。所以，我们在左图点一下，就在宇宙中发射了一射线；在右图点一下，又发射了一条射线。
理想情况下，这两条射线会在避雷针的尖端完美相交。
现实情况下，因为镜头有畸变、风吹着无人机抖动、鼠标点得不准，这两条射线在三维空间中其实是**“异面直线”（永远不会相交）**！3. 最小二乘法的降维打击
